ブラウザ上で動くライフゲーム(Conway's Game of Life)のシミュレーターです
（PC推奨）

## 操作方法

### grid
- グリッド上のエリアをクリックすることでセルの色を反転
- グリッド下の Start/Stop でシミュレーションを開始/停止
- Resetでグリッドを初期状態にリセット
- Rotateで選択しているテンプレートを90°回転

### Templates
- 画面右のエリアから何種類かのテンプレートを利用できます

### Settings
- Speed: シミュレーションの実行速度を調節できます　数値が小さいほど実行速度が速くなります
- Treat cells outside the grid as dead cells: 画面端を壁として扱うか、もう片方の端と繋げるかを選択できます
